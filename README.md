### 目的
设计一款更人性的运维平台  

* 如果你想找运维平台的设计思路，那么它就是一个设计思路
* 如果你想开发一套自己的运维平台，那么它就是一个开发起点
* 如果你想学习python/django开发，那么它就是一个开发教程

> 项目中有关IT资源都是基于阿里云的，如果要接入其它平台，我认为很容易修改(即使是混合云)
